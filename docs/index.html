Documentation

Installable OGN/FLARM receiver manager for Raspberry Pi.

Back to all Caen Falaise Planeurs projects

Key features:

  • Installs and manages an Open Glider Network receiver on an existing Raspberry Pi OS system
  • Explicit privileged setup with Debian dependencies, verified OGN binary downloads, and systemd units
  • Pi Connect-friendly CLI for status, logs, aircraft, config validation, and service checks
  • Pinned binary manifest with SHA-256 verification and documented third-party binary boundaries
  • Field-oriented quick start for confirming local aircraft reception
  • Quick Start
    Minimal commands for checking whether the receiver sees aircraft
  • Installation
    Pi install commands, package changes, setup, and uninstall
  • CLI Reference
    Status, aircraft, logs, service, config, and health commands
  • Operations

  • OGN Binaries

  • Troubleshooting
